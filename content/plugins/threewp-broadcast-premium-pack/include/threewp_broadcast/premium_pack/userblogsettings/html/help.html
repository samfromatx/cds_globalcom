Base concepts

The broadcast meta box is the Broadcast box that is seen when editing posts. User & Blog Settings (U&B) works by modifying (1) how Broadcast is shown to the user and (2) the meta box. The Super Admin creates modifications, that contains information about which parts of Broadcast should be modified. [caption id="attachment_264" align="aligncenter" width="300"] A modification[/caption] Each modification has one or more criteria, that decide which blog + role + user combinations the respective modification should apply to. [caption id="attachment_263" align="aligncenter" width="300"] Modification criteria[/caption]

Modifications

Each modification contains the following information:
  • The modification name, that should describe to the super admin what the modification does.
  • Visibility settings: which parts of Broadcast should be shown.
  • Meta box modifications: U&B automatically detects the settings available in the meta box and shows a selection of possible changes for each.
 

Criteria

The criteria decide which combinations of blogs, roles and users the modification applies to. When searching for applicable modifications, U&B searches for criteria in the following priority:
  1. A specific user on this blog
  2. A specific user with this role
  3. A specific user
  4. A role on this blog
  5. A blog
  6. A role
If a criteria match is found, any lower priorities are ignored: is a modification for a specific user + role (priority 2) is found, a modification with priority 4, "user on blog" is ignored even though it might be the same user on the correct blog. After the above, all modifications set to any blog, any role, any user will then be applied. Use the test function to see how modifications are selected.

How to create a modification

Before creating a modification, formulate a descriptive sentence that explains what the goal of the modification is: you want to do something, depending on specific criteria.
"I want to hide the Broadcast box from all editors on the blog Broadcast 3."
  1. Create a modification.
  2. Edit the modification.
  3. Write a description: Hide the broadcast box
  4. Uncheck Display Broadcast meta box.
  5. Save the modification.
  6. Go back to the modifications overview.
  7. Hover over the newly-created modification.
  8. Click on the Edit the criteria link.
  9. There should already by a row of criteria.
  10. Select the blog Broadcast 3.
  11. Save the criteria.
Or
"All authors should have their posts automatically posted and linked to the IT blog, no matter which blog they're writing on."
  1. Create a modification.
  2. Edit the modification.
  3. Write a description: Post and link to IT
  4. In the meta box settings:
  5. Find the link option.
  6. Change it to force on or force on and hide, depending on your preference.
  7. In the Broadcast to settings:
  8. Find the IT blog.
  9. Change it to force on or force on and hide, depending on your preference.
  10. Save the modification.
  11. Go back to the modifications overview.
  12. Hover over the newly-created modification.
  13. Click on the Edit the criteria link.
  14. There should already by a row of criteria.
  15. Select the role Author.
  16. Save the criteria.
Now you should test the modification to see if it is applied correctly.

Testing

The test tab can help show which modifications affect specific combinations of blogs / roles /users. Just select a combination and press the button. U&B will then show a list of the modifications that are applied to the selected combination. This feature helps to find conflicts between modifications. See the criteria search order, above.

FAQ

Is the super admin affected by modifications?

Yes, with one exception: the broadcast menu is never hidden from the super admin. How would one change U&B settings if one can not use the Broadcast menu? :)
