The Send To Many function appears as a button above the post list. Usage instructions:
  • Find the posts you want to broadcast
  • Check the box of each post
  • Press the Send To Many button. A meta box should appear above the post list.
  • Modify the broadcast settings.
  • Select which blogs you want the post broadcasted to.
  • Press the Send To Many button and wait.
Depending on how much data is to be broadcasted it could take up to 30 seconds to broadcast the posts to the blogs. If you have the Queue plugin active the page will immediately refresh and you can watch the broadcast progress for each post.
